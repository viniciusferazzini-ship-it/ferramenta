# Ferramenta de Preços

A full-stack pricing tool application built with a modern monorepo structure.

## Project Structure

```
ferramenta/
├── packages/
│   ├── api/              # Backend API (Node.js/Express)
│   ├── web/              # Frontend (React/Next.js)
│   └── shared/           # Shared types and utilities
├── .github/
│   └── copilot-instructions.md
├── docker-compose.yml
├── package.json
└── pnpm-workspace.yaml
```

## Quick Start

```bash
# Install dependencies
pnpm install

# Start development
pnpm dev
```
